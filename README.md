## Independent Projects

### StoreBanao - Multi-Vendor E-Commerce SaaS

[![Live](https://img.shields.io/badge/Live-storebanao.in-5ddba8?style=flat-square)](https://storebanao.in)
&nbsp;
![Next.js](https://img.shields.io/badge/Next.js_14-000000?style=flat-square&logo=nextdotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178c6?style=flat-square&logo=typescript&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-336791?style=flat-square&logo=postgresql&logoColor=white)
![Clerk](https://img.shields.io/badge/Clerk-6c47ff?style=flat-square&logo=clerk&logoColor=white)
![Turborepo](https://img.shields.io/badge/Turborepo-EF4444?style=flat-square&logo=turborepo&logoColor=white)

A production SaaS enabling Indian D2C sellers to launch and run online stores with zero technical knowledge. Designed, architected, and shipped end to end - from zero to live users in under 3 months.

**What makes it production-grade:**
- **Turborepo monorepo** with separate packages for storefront, vendor dashboard, and admin panel - clean separation, independent deployments
- **Clerk multi-tenant auth** with cross-subdomain session sharing - custom `isWebhookRoute` middleware fix resolving a 307 redirect conflict that was silently swallowing webhook events
- Full product, order, and inventory management with **Cloudinary CDN** - `public_id` stored in DB enabling clean image deletion via destroy API
- **Multi-language i18n system** with Bodo (brx-IN) language support - JSON locale files, lazy-loaded translations, localStorage persistence, SEO-friendly language-based routing
- Manual UPI billing live - Razorpay subscription billing system architected across phases
- Transactional emails via **Resend** - order confirmations, seller onboarding, and system alerts

### Planningo - All-in-One Productivity Platform

[![Live](https://img.shields.io/badge/Live-mydailyworkspace.site-5ddba8?style=flat-square)](https://mydailyworkspace.site)
&nbsp;
![Next.js](https://img.shields.io/badge/Next.js-000000?style=flat-square&logo=nextdotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178c6?style=flat-square&logo=typescript&logoColor=white)
![Supabase](https://img.shields.io/badge/Supabase-3ECF8E?style=flat-square&logo=supabase&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-336791?style=flat-square&logo=postgresql&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-0f172a?style=flat-square&logo=tailwindcss&logoColor=38bdf8)

A cloud-based productivity platform replacing spreadsheet chaos with six tightly integrated modules - all synced in real time across every device.

| Module | What it does |
|---|---|
| Todos | Prioritized tasks with due dates, tags, and statuses |
| Calendar | Monthly and weekly view with color-coded events |
| Day Planner | Drag-and-drop time-blocking with color coding |
| Reminders | Push notifications for important moments |
| Trip Planner | Multi-day itineraries with stops, notes, and budgets |
| Expenses and Budget | Group expense splitting and personal budget tracking |

**What makes it production-grade:**
- **Row-level security via Supabase** - enterprise-grade per-user data isolation with no server-side filtering logic required
- **Real-time sync** across phone, tablet, and desktop - no native app, pure web
- Drag-and-drop day planner with time-block scheduling and color-coded calendar events
- Shared group expense splitting and multi-day trip itinerary builder for real-world collaborative use cases
- Mobile-first responsive design - works identically across all screen sizes
